# Spotify Genre Segmentation and Recommendation System

## Project Overview
This project analyzes Spotify songs, performs genre segmentation using K-Means clustering, and builds a music recommendation system using cosine similarity.

## Features
- Data Preprocessing
- Exploratory Data Analysis
- Correlation Matrix
- K-Means Clustering
- PCA Visualization
- Genre-wise Cluster Analysis
- Playlist-wise Cluster Analysis
- Song Recommendation System

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Dataset
Spotify Songs Dataset

## How to Run

pip install pandas numpy matplotlib seaborn scikit-learn

python spotify_project.py
